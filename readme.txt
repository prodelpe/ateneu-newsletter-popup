Popup de subscripció a la newsletter integrat amb Benchmark Email.

== Description ==

Mostra un popup de subscripció al visitant del web. S'integra amb Benchmark Email: enganxes el codi del formulari tipus "Popup" de Benchmark i el plugin s'encarrega de:

* Carregar el script respectant una cookie (per no mostrar el popup cada visita).
* Amagar les etiquetes dels camps mantenint l'accessibilitat.
* Substituir els missatges d'error nadius per un banner inline traduït al català.

Configuració a **Newsletter Popup** del menú d'admin.
